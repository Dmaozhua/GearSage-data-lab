# Confidence rules

## high
The value is explicitly stated in:
- spoken audio
- subtitle text
- visible on-screen text
- title or description

## medium
The value is strongly supported by context, repeated indirectly, or derived from a clear comparison relation.

## low
The value is weakly supported, ambiguous, or inferred from incomplete evidence.

## Default review rule
All numeric fields and material fields should default to review_required=yes.
